Jifbox: gif photobooth

This was a fun side project done in a relatively short timeframe for Sunlight's 2014 Halloween Open House. In previous years, we had a photobooth app that would produce a digital photostrip. This project replaced that photobooth app with… GIFs. Jifbox is a web-based animated GIF photobooth. GIFs are generated in the browser using your webcam and an interactive interface.

I led user experience and designed interactions for the webapp as well as the physical interactive piece we made for the actual photobooth at the event. The pièce de résistance: A big red button mounted inside a skull that triggered the GIF making.
